# Property Buying Process Knowledge - Alaya Framework

A comprehensive knowledge base showcasing a data-driven framework for property investment, built on insights from Alaya Property and industry experts.

## Overview

This website presents a structured approach to property buying, organized into three main categories:
- **THE MACRO** - Macroeconomic analysis and sources
- **THE MICRO** - Micro-level research tools and market indicators
- **THE ASSET** - Property-specific analysis and deal evaluation

## Features

- Interactive accordion sections for easy navigation
- Responsive design optimized for mobile and desktop
- Data-driven property analysis framework
- Comprehensive source references for research

## Technology

- Pure HTML, CSS, and JavaScript
- Google Fonts (Montserrat & Open Sans)
- No external dependencies required

## Setup

1. Clone this repository:
```bash
git clone https://github.com/yourusername/property-buying-knowledge.git
```

2. Open `property-buying-knowledge.html` in your web browser

That's it! No build process or dependencies needed.

## Required Assets

Make sure the following image files are in the same directory as the HTML file:

- `New Header Image 1.jpg` - Hero section background image
- `profile picture.png` - Profile image
- `new brain.png` - Brain icon for learnings section
